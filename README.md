## 🏁 Getting Started

### 🔧 Prerequisites
This project requires:

- Node.js  
- TypeScript  
- npm  

---

## 📥 Installation

### 1. Clone the repository
```sh
git clone https://github.com/AmierKhalid/Code-Zone
```
### 2. Move to the folder:
```sh
npm install
```
###▶️ Usage
```sh
npm run dev
```
